About

Welcome to the official Playit Backend API documentation. This API is designed to provide robust backend services for media-centric applications. Explore our endpoints to integrate features like user authentication, media management, playlists, comments, likes, subscriptions, and real-time updates into your app.

Key Features

  • User Authentication: Secure user registration and login.
  • Media Management: Upload, fetch, and delete media files.
  • Playlist Management: Create, update, and manage playlists.
  • Comments and Likes: Interact with video content using comments and likes.
  • Subscriptions: Manage user subscriptions and notifications.
  • Real-Time Updates: WebSocket support for instant updates.
